# DBMS Practical Website

##  Live Demo

https://mohit-jagudi.github.io/dbms-practical-website/

A simple and interactive **web-based DBMS Practical Website** that allows students to learn, practice, and visualize SQL queries with questions, queries, and outputs in one place.

---

## Features

* Topic-wise SQL queries (DDL, DML, DQL, etc.)
* Interactive question selection system
* Displays question, SQL query, and output together
* Real-time output visualization in table format
* Clean and modern UI (glassmorphism design)
* Sidebar navigation for easy access to topics
* Beginner-friendly interface for DBMS learning
* Fully responsive design
* Organized SQL data using JavaScript

## How to Run

1. Download the project files (ZIP or manually)

2. Create a new folder on your system
   Example: `dbms-practical-website`

3. Move all project files into that folder:

   * `index.html`
   * `programs.html`
   * `script.js`
   * `styles.css`

4. Open this folder in any code editor (like VS Code)

5. Open `index.html` in browser

OR

Right click → Open with browser

---

## How to Use

1. Open the website

2. Navigate between pages:

   * About Page (Project Details)
   * Programs Page (SQL Queries)

3. Select any topic from sidebar:

   * DDL Commands
   * DML Commands
   * SELECT Queries
   * WHERE, ORDER BY, ALIAS
   * LIKE Operator
   * GROUP BY / HAVING
   * Subqueries
   * Joins
   * PL/SQL

4. Click on any question

5. View:

   * Question
   * SQL Query
   * Output Table
